Driver Vehicle License Department ( DVLD )

Driver and Vehicle Licensing System is a comprehensive system that issues both local and international licenses in accordance with established standards and conditions, as well as conducting the necessary tests for drivers. The system also allows scheduling appointments for medical exams or license collection. It provides services for license renewal, and issuing replacement licenses in case of loss or damage. Additionally, the system includes features for license suspension and release.

It was used in the project

  • User Interface – Developed using Windows Forms in C# .Net Framework 4.8.1
  • Database – SQL Database is used for storing and managing data.
  • Architecture – Three-layer architecture(Presentation Layer (UI), Business Logic Layer, Data Access Layer).
  • User Controls – User Controls are utilized to create reusable components for a modular and maintainable design.
  • Password Encryption – Implemented password encryption for secure user authentication.

Core Management Features

  • 1. Local Driver License Management – Add, Edit, Cancel, Issue Licenses, Display License Information, Conduct and Manage Driving Tests.
  • 2. Application Management – Add, Edit, Cancel, and Display Applications.
  • 3. User Management – Add, Edit, Display, Delete Users, Change Passwords, Manage Permissions.
  • 4. People Management – Add, Edit, Display, Delete Records, Ensure No Duplicate Entries.
  • 5. Application Type Management – Search, Display, Edit, and Filter Application Types.
  • 6. Test Type Management – Search, Display, Edit Test Prices, and Apply Filters.
  • 7. License Type Management – Edit Requirements for Different License Types.
  • 8. Detained Licenses Management – Track Selected Licenses with Law and Reservation Dates.
