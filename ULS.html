University Library System - ULS

The University Library System (ULS) is a comprehensive system designed to manage university libraries efficiently. It facilitates book archiving, borrowing, and returning processes while keeping track of book availability and user activities.

It was used in the project

  • User Interface – Developed using Windows Forms in C# .Net Framework 4.8.1
  • Database – SQL Database is used for storing and managing data.
  • Architecture – Three-layer architecture(Presentation Layer (UI), Business Logic Layer, Data Access Layer).
  • User Controls – User Controls are utilized to create reusable components for a modular and maintainable design.
  • Password Encryption – Implemented password encryption for secure user authentication.

Core Management Features

  • 1. Borrowing Records – Track borrowed books, manage due dates, and send overdue notifications.
  • 2. Fine Records – Calculate and track fines for overdue books, generate reports, and manage adjustments.
  • 3. Students Management – Register students, track borrowing history, and manage access permissions.
  • 4. Book Management – Add, edit, delete, and categorize books for easy retrieval.
  • 5. Library Visits Tracking – Log and analyze student and faculty visits to the library.
  • 6. College & Majors Management – Organize books and resources by academic disciplines.
  • 7. Research & Publications – Store, categorize, and provide access to research papers and theses.

Additional Information

  • Total Forms: 36
  • User Controls: 9
  • Classes: 45
  • Development Duration: one Month
